# Amino Acid Age Prediction Models

This repository contains scripts for building and evaluating multiple machine learning models to predict chronological age from amino acid concentrations.

## Project Overview

- **Objective**: Predict chronological age using plasma amino acid concentrations and compare multiple machine learning algorithms.  
- **Dataset**: Processed dataset includes amino acid features and chronological age (`age_year`).  
- **Models**: Ten models were trained and evaluated:
  1. Lasso Regression
  2. Elastic Net
  3. Linear Regression
  4. Decision Tree
  5. K-Nearest Neighbors (KNN)
  6. Random Forest
  7. XGBoost
  8. LightGBM
  9. Support Vector Machine (SVM)
  10. Neural Network

- **Evaluation Metrics**:
  - RMSE (Root Mean Squared Error)
  - MAE (Mean Absolute Error)
  - R² (Coefficient of Determination)
